Game — 轻量游戏框架— lightweight game frameworks

Game.Foundation

渲染无关的固定步长计时、确定性随机、语义化输入、场景生命周期；Game.Foundation.Sdl 提供可选的 SDL 应用宿主。Renderer-neutral fixed-step timing, deterministic random, semantic input, scene lifecycle; Game.Foundation.Sdl is an optional SDL host.

Game.Board · Game.Cards · Game.Arcade2D

Board：可克隆网格、寻路、回合、校验命令、快照与回放。Cards：卡牌目录、运行期实例、确定性区域、构筑对战运行时。Arcade2D：几何、碰撞查询、动画状态、对象池、路径跟随。 Board: cloneable grids, pathfinding, turns, validated commands, snapshots & replay. Cards: catalogs, runtime instances, deterministic zones, deck-building battle runtime. Arcade2D: geometry, collision queries, animation state, pooled entities, path following.

Game.Zgm (ZanGameMaker) · Game.Arpg

Zgm：带项目校验的类型化组件模型/运行时——生命周期与事件、地图、角色、道具、技能、Buff、窗口、控件、补间、存档快照、TCP/WebSocket 联网与 SQLite。Arpg：独立的类型化 RPG 运行时——校验的项目数据、实时世界、战斗、调度、图形控件、数据绑定与 SDL 渲染。 Zgm: a typed component model/runtime with project validation — lifecycle & events, maps, roles, items, skills, buffs, windows, widgets, tweens, save snapshots, TCP/WebSocket networking and SQLite. Arpg: a separate typed RPG runtime — validated project data, live world, combat, scheduling, graphical controls, data binding and SDL rendering.

SDL3 / Platform

SDL3 负责平台窗口、渲染、输入与计时；Platform 提供跨平台底层适配。游戏框架的 SDL 宿主构建于此。 SDL3 owns platform windows, rendering, input and timing; Platform provides the cross-platform layer. The game SDL host builds on this.
