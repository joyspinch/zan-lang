💡 编译器与标准库都已内置在 IDE 里，无需另外安装任何东西。 The compiler and standard library are bundled in the IDE — nothing else to install.

安装与首次启动Install & first launch

从下载页取得压缩包（提取码 zanlang），解压到任意目录，双击可执行文件即可启动。绿色免安装，不写系统注册表、不改 PATH。 Grab the archive from the download page (access code zanlang), unzip anywhere and run the executable. Portable — no registry writes, no PATH changes.

  • Windows：解压后运行 ZanIDE.exe。Windows: run ZanIDE.exe.
  • Linux：chmod +x ZanIDE 后运行。Linux: chmod +x ZanIDE then run.
  • macOS：拖入应用程序，首次运行右键「打开」。macOS: move to Applications, first run via right-click → Open.

新建项目与模板New project & templates

菜单 File → New Project，选择一个模板，填项目名，确定。IDE 会生成完整可运行的项目（含 zan.proj、源码、示例视图）。 Use File → New Project, pick a template, name it, confirm. The IDE scaffolds a fully runnable project (with zan.proj, sources and sample views).

编辑器与项目树Editor & project tree

左侧是项目树，中间是多标签编辑器，底部是输出面板。编辑器提供语法高亮、自动补全、跳转定义、实时错误提示。 Project tree on the left, a tabbed editor in the middle, output panel at the bottom. The editor offers highlighting, autocomplete, go-to-definition and live diagnostics.

运行 / 构建 / 发布Run / build / publish

这些操作都是项目级的——始终作用于当前项目（由 zan.proj 定义的入口 + 全部源文件），与你当前正打开、聚焦哪个文件无关。 These are project-level — they always target the current project (the entry + all sources declared by zan.proj), regardless of which file is focused.

.zform 可视化设计器.zform visual designer

.zform 是可视化设计文件。打开它后，可在同一个标签内切换「设计视图 ↔ 代码视图」——切换只是换个视图看同一个文件，不会新建文件、也不会新开标签。 A .zform is a visual design file. Inside the same tab you toggle Design ↔ Code — it's just two views of the one file; it never creates a new file or tab.

  • 拖拽控件布局，属性面板改样式Drag controls to lay out, tweak style in the property panel
  • 切到代码视图查看/编辑对应代码，运行为原生窗口Switch to Code to view/edit the code; run as a native window
  • 只有 .zform 文件才有设计/代码切换；普通 .zan 文件该按钮为禁用状态Only .zform files can switch; for ordinary .zan files the button is disabled

调试与断点Debug & breakpoints

在行号左侧点击设置断点，按 F5 以调试方式运行；命中断点后可查看变量、单步执行、继续。 Click the gutter to set breakpoints, press F5 to run in debug; inspect variables, step and continue when a breakpoint hits.

热重载Hot reload

热重载只在运行/调试会话进行时可用：会话运行中保存文件即触发重建并刷新。平时（未运行时）热重载按钮为禁用状态，避免误操作。 Hot reload is available only during a run/debug session: while running, saving a file triggers a rebuild and refresh. When not running, the hot-reload button is disabled to prevent mistakes.

发布与跨平台Publish & cross-compile

发布对话框的目标始终是当前项目（不是当前打开的某个源文件）。选择目标平台即可交叉编译出单文件产物，拷贝到目标机器即可运行。 The Publish dialog always targets the current project (not whichever file is open). Pick a target platform to cross-compile a single-file build you can copy and run.

AI 助手 & RepoMapAI assistant & RepoMap

IDE 内置 AI 助手，在设置的「AI 助手」区可配置接口地址 / 模型 / API Key（保存到 ai.cfg）。助手会把 RepoMap 代码库大纲 + 当前文件作为上下文，实现「读懂整库、精准改代码」。 The IDE ships a built-in AI assistant; configure endpoint / model / API key in Settings → AI (saved to ai.cfg). It feeds the RepoMap repo outline plus the current file as context so the AI understands the whole repo and edits precisely.

  • .zanmap/repo.map.txt：文件 + 类 + 方法签名大纲.zanmap/repo.map.txt: files + classes + method signatures
  • symbols.json / routes.json：符号定位与路由表symbols.json / routes.json: symbol locations and route table
  • 开放 MCP 契约：repo://map · repo://symbols · repo://routesOpen MCP contracts: repo://map · repo://symbols · repo://routes

项目文件 zan.projThe zan.proj file

每个项目根目录都有一个 zan.proj，声明项目入口与要编译的源文件。运行/构建/发布都以它为准，所以无论聚焦哪个文件都能正确编译整个项目。 Every project root has a zan.proj declaring the entry and the sources to compile. Run/build/publish read it, so the whole project compiles correctly no matter which file is focused.

常见问题FAQ

需要先装 SDK / 配环境吗？Do I need to install an SDK / set up env?

不需要。编译器与标准库都在 IDE 里，下载解压即可用。No. The compiler and stdlib are inside the IDE — just download and unzip.

新建的 MVC 项目运行只弹空控制台？My MVC project shows an empty console?

请确认以项目方式运行（F5）。运行是项目级的，会加载 zan.proj 的入口，正常会打印监听地址而不是空控制台。 Run it as a project (F5). Runs are project-level and use the entry from zan.proj, so you should see the listening address, not an empty console.

下载提取码是多少？What's the download code?

提取码是 zanlang，需在网盘页面手动粘贴（点站内「下载」按钮会弹出并可一键复制）。 The access code is zanlang, pasted manually on the netdisk page (the site's Download button pops it up to copy).
