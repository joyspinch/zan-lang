Hello World

程序从 Main 开始。Console.WriteLine 输出一行。 Programs start at Main. Console.WriteLine prints a line.

main.zan
using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, Zan!");
    }
}

命名空间 / usingnamespace / using

用 namespace 组织代码，用 using 引入其它命名空间。 Organize with namespace, import with using.

app.zan
using System;
using System.Collections;

namespace MyApp {
    class Service { }
}

变量：var / let / constVariables: var / let / const

  • var — 可变变量，类型推断mutable, type inferred
  • let — 只读绑定（不可再赋值）read-only binding
  • const — 编译期常量compile-time constant
vars.zan
var count = 0;          // int
count = count + 1;
let name = "Zan";      // 只读read-only
const int MAX = 100;   // 常量constant
string msg = "typed";  // 显式类型explicit type

可空与空安全Nullability

T? 表示可空。用 ?. 安全访问、?? 提供默认值。 T? marks nullable. Use ?. for safe access and ?? for defaults.

null.zan
string? maybe = null;
int len = maybe?.Length ?? 0;

struct

值类型，赋值即复制，适合小数据。Value type, copied on assignment — good for small data.

struct.zan
struct Vec2 {
    public float X;
    public float Y;
    public float Length() { return Math.Sqrt(X*X + Y*Y); }
}

class

引用类型，支持字段、属性、方法、构造函数、继承。Reference type with fields, properties, methods, constructors and inheritance.

class.zan
class Animal {
    public string Name;
    public Animal(string name) { Name = name; }
    public virtual void Speak() { Console.WriteLine("..."); }
}
class Dog : Animal {
    public Dog(string n) : base(n) { }
    public override void Speak() { Console.WriteLine("Woof"); }
}

标签联合（Tagged union）Tagged unions

带负载的枚举，配合模式匹配，安全地表达「多选一」。Enums carrying payloads, matched by pattern — a safe "one of".

union.zan
union Result {
    Ok(int value),
    Err(string message),
}

switch (r) {
    case Ok(var v): Console.WriteLine(v); break;
    case Err(var m): Console.WriteLine(m); break;
}

ARC 内存与 weakARC memory & weak

Zan 用自动引用计数（ARC）管理内存：确定性析构、无 GC 停顿。用 weak 打破循环引用。 Zan uses ARC: deterministic cleanup, no GC pauses. Use weak to break reference cycles.

arc.zan
class Node {
    public Node? next;
    public weak Node? parent;   // 弱引用，不增加计数weak ref, no retain
}

async / await

async.zan
async string Fetch() {
    var body = await HttpClient.GetAsync("api.example.com", 443, "/");
    return body;
}

FFI / DllImportFFI / DllImport

用 DllImport 直接调用系统 API 与原生库；用 repr("C") 声明 C 兼容内存布局。 Call system APIs and native libs via DllImport; declare C-compatible layout with repr("C").

ffi.zan
[repr("C")]
struct Point { public int x; public int y; }

[DllImport("user32")]
extern int MessageBoxA(long hwnd, string text, string caption, int type);
