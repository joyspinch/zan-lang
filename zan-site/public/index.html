C# 语法LLVM 后端ARC 内存值语义直连 FFI
hello.zan
using System;

namespace HelloWorld;

struct Point {
    public float X;
    public float Y;

    public float Length() => Math.Sqrt(X * X + Y * Y);
}

class Program {
    static void Main(string[] args) {
        var p = Point { X = 3.0, Y = 4.0 };
        Console.WriteLine($"Length: {p.Length()}");
    }
}

注解式 Web 开发

服务端模板用 attribute 驱动路由，按目录约定生成，统一安全取参。

  • 属性声明动词与元数据：[Get] [Post] [Auth] [Rank] [Lock]
  • 路由按 controller 目录结构自动推导
  • 构建期 RouteGen 生成路由表，运行零反射开销
  • 统一安全取参与过滤：ctx.In / InInt / InText
ApiController.zan
// 收集数据：POST /api/gather，需鉴权、rank≥3、用户级锁
[Api(post, route="/api/gather", title="收集数据",
     auth, rank=3, lock="user")]
static void Gather(HttpContext ctx) {
    string name = ctx.InText("name");
    ctx.Json("{\"ok\":true}");
}

开始使用

从源码构建编译器（需 LLVM 17+）。

build.sh
# 克隆仓库
git clone https://github.com/joyspinch/zan-lang.git
cd zan-lang

# 构建编译器
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build .

# 编译并运行你的第一个程序
./zanc hello.zan --auto-stdlib && ./hello
