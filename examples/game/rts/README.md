# RTS 演示（红警风格即时战略）

GameKit 即时战略演示：左侧大地图战场 + 右侧指挥面板，接近经典 RTS 的标准玩法。

## 玩法

- 采矿经济：矿车自动采矿并回矿场结算，金币显示在顶部资源条
- 建造：右侧面板「建造」页排队建筑，完成后显示「就绪」，点击后在战场放置
- 造兵：「单位」页 9 种单位（步兵/火箭兵/工程师/坦克/装甲车/防空车/矿车/吉普/火炮）
- 电力系统：电厂供电，电力不足时生产减速
- 操作：左键点选/拖框多选，右键移动/攻击；方向键或屏幕边缘滚屏；F11 全屏
- 敌方 AI 会发展经济、生产部队并定期进攻；摧毁敌方基地获胜

## 素材

`assets/` 下的全部图片由 gpt-image-2 生成（绿幕精灵表 + 抠图切片），
蓝红两方用运行时染色（SetColor）区分，无版权素材。

## 构建

```
build/zanc examples/game/rts/main.zan examples/game/common/GameKit.zan --auto-stdlib -o build/rts.exe
```
